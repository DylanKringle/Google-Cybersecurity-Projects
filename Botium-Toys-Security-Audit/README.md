# Botium Toys Security Audit

## Overview

Completed a simulated organizational security audit for Botium Toys as part of the Google Cybersecurity Professional Certificate.

## Skills Demonstrated

- Security auditing
- Risk assessment
- Compliance review
- Security controls evaluation

## Frameworks Used

- NIST Cybersecurity Framework (NIST CSF)
- PCI DSS
- GDPR
- SOC Controls

## Files Included

- Scope, Goals, and Risk Assessment Report
- Controls and Compliance Checklist

## Outcome

Conducted a simulated organizational security audit by reviewing assets, evaluating security controls, assessing compliance requirements, and identifying areas for improvement.
