# Network Hardening Risk Assessment

## Overview

This project evaluates a simulated security breach at a social media organization and recommends network hardening practices to reduce future cybersecurity risks.

This assessment was completed as part of the Google Cybersecurity Professional Certificate.

---

## Scenario

A social media organization experienced a major data breach that exposed customer information.

The following vulnerabilities were identified:

- Employees shared passwords.
- The database administrator account used the default password.
- Firewall rules were not configured properly.
- Multi-factor authentication (MFA) was not enabled.

---

## Security Recommendations

Based on the assessment, I recommended:

- Port Filtering
- Strong Password Policies
- Multi-Factor Authentication (MFA)

---

## Skills Demonstrated

- Security risk assessment

- Security policy analysis

---

## What I Learned

This project reinforced the importance of implementing layered security controls.

Key takeaways include:

- Strong password policies reduce the risk of credential attacks.
- MFA adds an additional layer of authentication beyond passwords.
- Proper firewall configuration and port filtering reduce the attack surface.
- Security controls work best when implemented together rather than individually.

## Outcome

Completed a simulated network security risk assessment by identifying organizational vulnerabilities and recommending practical hardening techniques to improve the organization's security posture.
